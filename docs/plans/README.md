## 慣例
- 檔名：`YYYY-MM-描述.md`（例：`2026-07-improvement-roadmap.md`）。
- 每份計畫開頭寫 **Context**（為什麼做、要解決什麼、預期結果），再列分階段步驟與驗證方式。
- 狀態標記：`規劃中` / `進行中` / `已完成` / `擱置`。
